******* Инструкция по сборке проекта *******

В папке:
- ConsoleFTPclient расположен исходный java код программы;
- в папке test расположен исходный код тестов;
Их которые легко можно импортировать в IntelliJ IDEA для сборки проекта
под любую операционную систему (например Linux).

*** СБОРКА ЧЕРЕЗ КОНСОЛЬ ***
Cпособ сборки через командную строку (консоль):

Из папки ../src/main/java/ConsoleFTPclient/ где расположены все файлы проекта в формате *.java:
1. Перемещаем их в новую корневую папку, например FTPClient.
2. Запускаем консоль.
3. Переходим в подготовленную папку с проектом, например ../FTPClient/ConsoleFTPclient.
4. Запускаем в терминале команду:

javac -classpath . MainApp.java jsonconverter/*.java menuitems/*.java services/*.java -d ../out

5. В папке FTPClient появится папка out со скомпилированными файлами *.class проекта.
6. В папке ../META-INF/ расположен файл манифеста который нам тоже нужен для сборки.
7. Копируем папку META-INF в папку out.

Теперь в папке ../FTPClient/out содержатся наш скомпилированный проект
и манифест, где указан запускаемый файл MainApp.

8. Переходим в консоли в папку out. Вводим и запускаем следующую строку:

Jar -cfm FTPClient.jar META-INF/MANIFEST.MF ConsoleFTPclient/*.class ConsoleFTPclient/jsonconverter/*.class ConsoleFTPclient/menuitems/*.class ConsoleFTPclient/services/*.class

9. Если параметры среды настроены правильно, то в папке out появится файл с расширением *.jar, в нашем случае FTPClient.jar, см. командную строку выше.

Файл запускается стандартными средствами java.

*** СБОРКА СРЕДСТВАМИ IntelliJ IDEA ***
Данный проект можно собрать средствами MAVEN, либо средствами IntelliJ IDEA
используя графический интерфейс программы. В данной инструкции этот способ
не рассматривается.
